Fast floating point math library and extensions

qLibs provides its own math library with a set of APIs to compute common mathematical functions and transformations. The qlibs::ffmath can replace a default math floating-point library, delivering better performance with less code delivering a very close aproximations to the standard library. qlibs::ffmath is very fast and small, it delivers FPU-like performance in pure software. Where available, it even boosts the performance of an FPU for complex mathematical functions.

The list of provided functions and constants:

Classification

  • qlibs::ffmath::classify() Categorizes the floating-point number
  • qlibs::ffmath::isNan() Determine if argument is Not-A-Number (NaN)
  • qlibs::ffmath::isInf() Determine if argument is Infinity.
  • qlibs::ffmath::isFinite() Determines if the given floating point number has finite value.
  • qlibs::ffmath::isNormal() Determines if the given floating point number is normal

Basic functions

  • qlibs::ffmath::sign() Computes the sign function
  • qlibs::ffmath::absf() Computes the absolute value
  • qlibs::ffmath::recip() Computes the multiplicative inverse or reciprocal
  • qlibs::ffmath::sqrt() Computes the square-root
  • qlibs::ffmath::rSqrt() Computes the reciprocal square-root
  • qlibs::ffmath::cbrt() Computes the cubic-root
  • qlibs::ffmath::rCbrt() Computes the reciprocal cubic-root
  • qlibs::ffmath::rounding() Computes the nearest integer rounding halfway cases away from zero.
  • qlibs::ffmath::floor() Computes the largest integer value not greater than the given argument.
  • qlibs::ffmath::ceil() Computes the smallet integer value not greater than the given argument.
  • qlibs::ffmath::trunc() Computes the nearest integer not greater in magnitude than the given argument.
  • qlibs::ffmath::frac() Obtain the fractional part
  • qlibs::ffmath::rem() Computes the floating point remainder after division
  • qlibs::ffmath::mod() Computes the floating point remainder after division

Trigonometric functions

  • qlibs::ffmath::sin() Computes the sine in radians
  • qlibs::ffmath::cos() Computes the cosine in radians
  • qlibs::ffmath::tan() Computes the tangent in radians
  • qlibs::ffmath::asin() Computes the principal value of the arc sine
  • qlibs::ffmath::acos() Computes the principal value of the arc cosine
  • qlibs::ffmath::atan() Computes the principal value of the arc tangent
  • qlibs::ffmath::atan2() Computes the arc tangent
  • qlibs::ffmath::sinh() Computes hyperbolic sine
  • qlibs::ffmath::cosh() Computes hyperbolic cosine
  • qlibs::ffmath::tanh() Computes hyperbolic tangent
  • qlibs::ffmath::asinh() Computes the inverse hyperbolic sine
  • qlibs::ffmath::acosh() Computes the inverse hyperbolic cosine
  • qlibs::ffmath::atanh() Computes the inverse hyperbolic tangent
  • qlibs::ffmath::wrapToPi() Wraps angle in radians, to the interval [−pi, pi]
  • qlibs::ffmath::wrapTo2Pi() Wraps angle in radians, to the interval [0, 2*pi]
  • qlibs::ffmath::wrapTo180() Wraps angle in degrees, to the interval [−180, 180]
  • qlibs::ffmath::wrapTo360() Wraps angle in degrees, to the interval [0, 360]

Exponential functions

  • qlibs::ffmath::exp2() Computes 2 raised to the given power
  • qlibs::ffmath::log2() Computes the base 2 logarithm
  • qlibs::ffmath::exp() Computes the e (Euler's number) raised to the given power
  • qlibs::ffmath::exp10() Computes the value of 10 raised to the power
  • qlibs::ffmath::log() Computes the natural (base e) logarithm
  • qlibs::ffmath::log10() Computes the common (base-10) logarithm
  • qlibs::ffmath::pow() Computes the power raised to the base number

Special functions

Most special functions are designed to prioritize precision over speed given their applications, and their implementation closely follows that of the C++ standard. qlibs::ffmath provides these special functions to ensure compatibility with compilers that lack them.

  • qlibs::ffmath::erf() Computes the error function
  • qlibs::ffmath::erfc() Computes the complementary error function
  • qlibs::ffmath::rexp() Decomposes given floating point value into a normalized fraction and an integral power of two.
  • qlibs::ffmath::ldexp() Multiplies a floating point value by the number 2 raised to the power.
  • qlibs::ffmath::hypot() Computes the square root of the sum of the squares
  • qlibs::ffmath::nextAfter() Compute the next representable floating-point value following x in the direction of y
  • qlibs::ffmath::tgamma() Computes the gamma function
  • qlibs::ffmath::lgamma() Computes the natural logarithm of the gamma function
  • qlibs::ffmath::factorial() Computes the factorial of the integer part
  • qlibs::ffmath::assoc_laguerre() Computes the associated Laguerre polynomials
  • qlibs::ffmath::assoc_legendre() Computes the associated Legendre polynomials
  • qlibs::ffmath::beta() Computes the Beta function
  • qlibs::ffmath::comp_ellint_1() Computes the complete elliptic integral of the first kind
  • qlibs::ffmath::comp_ellint_2() Computes the complete elliptic integral of the second kind
  • qlibs::ffmath::comp_ellint_3() Computes the complete elliptic integral of the third kind
  • qlibs::ffmath::ellint_1() Computes the incomplete elliptic integral of the first kind
  • qlibs::ffmath::ellint_2() Computes the incomplete elliptic integral of the second kind
  • qlibs::ffmath::ellint_3() Computes the incomplete elliptic integral of the third kind
  • qlibs::ffmath::expint() Computes the Exponential integral
  • qlibs::ffmath::hermite() Computes the (physicist's) Hermite polynomials
  • qlibs::ffmath::laguerre() Computes the non-associated Laguerre polynomials
  • qlibs::ffmath::legendre() Computes the unassociated Legendre polynomials
  • qlibs::ffmath::riemann_zeta() Computes the Riemann zeta function
  • qlibs::ffmath::sph_bessel() Computes the spherical Bessel function of the first kind
  • qlibs::ffmath::sph_neumann() Computes the spherical Bessel function of the second kind ( spherical Neumann function )
  • qlibs::ffmath::cyl_bessel_i() Computes the regular modified cylindrical Bessel function
  • qlibs::ffmath::cyl_bessel_j() Computes the cylindrical Bessel function of the first kind
  • qlibs::ffmath::cyl_bessel_k() Computes the irregular modified cylindrical Bessel function
  • qlibs::ffmath::cyl_neumann() Computes the cylinkdrical Neumann function.
  • qlibs::ffmath::sph_legendre() Computes the spherical associated Legendre function

Constants

  • qlibs::ffmath::getNan() Returns Not a Number (NaN) nan as a 32-bit floating point number
  • qlibs::ffmath::FFP_E The base of natural logarithms ( e )
  • qlibs::ffmath::FFP_LOG2E The base 2 logarithm of e
  • qlibs::ffmath::FFP_LOG10E The base 10 logarithm of e
  • qlibs::ffmath::FFP_LN2 The natural logarithm of 2
  • qlibs::ffmath::FFP_LN10 The natural logarithm of 10

  • qlibs::ffmath::FFP_PI_2 Half of π
  • qlibs::ffmath::FFP_PI_4 A quarter of π

  • qlibs::ffmath::FFP_2_PI Twice the inverse of π
  • qlibs::ffmath::FFP_2_SQRTPI The inverse of the square root of π
  • qlibs::ffmath::FFP_SQRT2 The square root of 2
  • qlibs::ffmath::FFP_SQRT1_2 The inverse of square root of 2

Math extra

  • qlibs::ffmath::lerp() Linear interpolation
  • qlibs::ffmath::midpoint() Computes the midpoint of two floating-points
  • qlibs::ffmath::map() Scales the given input from a range to another
  • qlibs::ffmath::normalize() Normalize the given input to value range between 0 and 1.
  • qlibs::ffmath::inRangeCoerce() Determines if a value falls within a range and coerces the value to fall within the range
  • qlibs::ffmath::isEqual() Determines if the parameters given as floating-point values are approximately equal.
  • qlibs::ffmath::inPolygon() Determines if a point is inside a polygon
  • qlibs::ffmath::inCircle() Determines if a point is inside a circle
