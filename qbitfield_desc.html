qlibs::bitfield class contains a setof functions for creating and manipulating bit fields also known as bit arrays. These are series of zeroes and ones spread across an array of storage units, such as unsigned long integers.

Structure

Bit arrays are stored in the qlibs::bitfield class. This class has two elements: an array of unsigned long integers called field for storing the bits and an integer size for storing the number of bits in the array.

Functions

Instance setup

  • qlibs::bitArea Use it to instantiate the area to store a BitField.
  • qlibs::bitfield::setup() Setup a initialize a BitField instance.

Single-bit

  • qlibs::bitfield::setBit() Sets one bit in a BitField.
  • qlibs::bitfield::clearBit() Clears one bit in a BitField.
  • qlibs::bitfield::toggleBit() Toggles (i.e. reverses the state of) a bit in a BitField.
  • qlibs::bitfield::readBit() Retrieve the state of a bit in a bitfield.
  • qlibs::bitfield::writeBit() Writes one bit in a bitfield.

Multiple-bits

  • qlibs::bitfield::clearAll() Clear all the bits in the BitField.
  • qlibs::bitfield::setAll() Set all the bits in the BitField.
  • qlibs::bitfield::readUINTn() Reads an unsigned 32-bit value from the BitField.
  • qlibs::bitfield::writeUINTn() Writes an unsigned n-bit value from the BitField.
  • qlibs::bitfield::readFloat() Reads a 32-bit floating point value from the BitField.
  • qlibs::bitfield::writeFloat() Writes a 32-bit floating point value to the BitField.

Dump

  • qlibs::bitfield::dump() Copies n bytes from the bit-field instance to a designed memory area.

Working with bitfields

#include <iostream>
#include <qlibs.h>
using namespace qlibs;
int main( int argc, char *argv[] )
{
bitfield vPort; // create the bitfield instance
bitArea<48> vPortArea; // Create the bitfield storage area to hold 48 bits
uint16_t rWord;
vPort.setup( vPortArea );
vPort.clearAll();
//we are going to write the following value in the bitfield = 0x8177AA55FF88
vPort.writeUINTn( 0, 0xAA55FF88, 32 ); // write the first 32 bits
vPort.writeUINTn( 32, 0x77, 8 ); // write one of the last two bytes
vPort.writeBit( 47, 1 ); // write the last bit of the last byte
vPort.writeBit( 40, 1 ); // write the first bit of the last byte
rWord = static_cast<uint16_t>( vPort.readUINTn( 20, 16 ); // read a word at offset 24
std::cout << std::hex << vPortArea[0] << ", "
<< vPortArea[1] << ", "
<< vPortArea[2] << ", "
<< vPortArea[3] << ", "
<< vPortArea[4] << ", "
<< vPortArea[5] << ", "
<< std::endl ;
return EXIT_SUCCESS;
}
